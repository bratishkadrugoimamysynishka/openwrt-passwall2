# Автоматический установщик Passwall 2 + Argon для OpenWrt (APK)

Скрипт `openwrt-passwall2.sh` предназначен для быстрой, безопасной и отказоустойчивой установки прокси-клиента **Passwall 2** и современной темы оформления **Argon** на роутеры под управлением **OpenWrt 25.12+**, использующие новый пакетный менеджер `apk`.

**Версия скрипта: v5**. В этой версии добавлена автоматическая очистка временных файлов и бэкапов после завершения установки, а также установлены новые настройки по умолчанию для минимального вмешательства в систему (Argon отключен, нет сетевого тюнинга).

---

## 🚀 Быстрая установка

Для запуска установки со стандартными настройками (Argon отключен, нет сетевого тюнинга) выполните в терминале роутера следующую команду:

echo "#############################################################"
echo "# Запуск с дефолтными настройками (v5):"
echo "rm -rf /tmp/openwrt-passwall2.sh ; wget -qO /tmp/openwrt-passwall2.sh \"https://raw.githack.com/bratishkadrugoimamysynishka/openwrt-passwall2/main/installer.sh\" ; sh /tmp/openwrt-passwall2.sh"
echo "#############################################################"

## ✨ Ключевые особенности скрипта

* **Архитектура нового поколения:** Полная совместимость с OpenWrt 25.12+ и менеджером пакетов `apk`.
* **Умный откат (Rollback):** Если что-то пойдет не так, скрипт автоматически откатит изменения.
* **Автоматическая очистка:** После завершения работы скрипт сам удаляет временные рабочие директории и созданные каталоги бэкапов (`/root/passwall2-backup-*`).
* **Безопасное скачивание:** Файлы скачиваются во временную директорию с атомарным перемещением. Никаких битых или недокачанных пакетов в системе.
* **Обход проблем со временем:** Встроенная мягкая синхронизация времени через `sysntpd`. Если время на роутере сбито (1970 год), доступен флаг `INSECURE_HTTPS=1`.
* **Зеркала GitHub:** Автоматический перебор встроенных рабочих зеркал в случае отказа основного API.
* **Автоматический бэкап:** Перед началом работы создается резервная копия критически важных конфигов (`system`, `network`, `firewall`, `luci`) в папку `/root/passwall2-backup-...`.

## 🛠️ Настройка и переменные окружения

Вы можете управлять поведением `installer.sh`, передавая переменные окружения перед командой запуска.

### Основные параметры (По умолчанию: Argon=0, Tuning=0)

| Переменная | По умолчанию | Описание |
| :--- | :---: | :--- |
| `INSTALL_ARGON` | `0` | Установить современную тему оформления **Argon** (`1` — да, `0` — нет). |
| `APPLY_SYSTEM_TUNING`| `0` | Применить оптимизацию сети и выставить часовой пояс (`1` — да, `0` — нет). |
| `AUTO_REBOOT` | `0` | Автоматически перезагрузить роутер после успешной установки (`1` — да). |
| `INSECURE_HTTPS` | `0` | Игнорировать проверку SSL-сертификатов (`1` — полезно, если на роутере совсем сбито время). |
| `GITHUB_PROXY_PREFIX`| `""` | Возможность указать свой собственный прокси-сервер для скачивания с GitHub. |

### Параметры тюнинга (при `APPLY_SYSTEM_TUNING=1`)

* **`TIMEZONE_NAME`** (по умолчанию `Europe/Moscow`): Имя часового пояса.
* **`TIMEZONE_STRING`** (по умолчанию `MSK-3`): Строка таймзоны для OpenWrt.
* **`ENABLE_PACKET_STEERING`** (`1`): Включение Packet Steering.
* **`ENABLE_FLOW_OFFLOADING`** (`1`): Программный ускоритель трафика (Flow Offloading).
* **`ENABLE_HW_FLOW_OFFLOADING`** (`0`): Аппаратный ускоритель трафика (включайте, только если ваш чипсет его поддерживает).

### Пример запуска с кастомными флагами:

Включить тему Argon, применить сетевые оптимизации и автоматически перезагрузить роутер по окончании:
echo "INSTALL_ARGON=1 APPLY_SYSTEM_TUNING=1 AUTO_REBOOT=1 sh /tmp/openwrt-passwall2.sh"
